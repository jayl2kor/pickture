## 앱 설명 (한국어)

비슷한 인물 사진이 여러 장 있을 때, 어떤 사진이 가장 잘 나왔는지 고르기 어려우셨나요?

Pickture가 AI 기술로 자동 분석하여 베스트 사진 3장을 선별해드립니다.

### 5가지 분석 기준

- 선명도: 사진이 얼마나 선명한지 분석
- 얼굴 품질: 얼굴이 자연스럽고 잘 나왔는지 평가
- 눈 뜨임: 눈을 감은 사진을 자동 감점
- 노출: 밝기가 적절한지 확인
- 구도: 인물 배치가 안정적인지 평가

### 주요 기능

- 4장 이상의 사진을 선택하면 자동으로 순위를 매겨줍니다
- 상위 3장은 상세 점수와 함께 큰 카드로 표시
- 인물 사진과 전신 사진 모드 지원
- 마음에 드는 사진을 즐겨찾기에 바로 등록
- 필요 없는 사진은 앱에서 바로 삭제
- 사진 비교 기능으로 두 사진을 나란히 비교

### 개인정보 보호

- 모든 분석은 기기에서만 수행됩니다
- 사진이 외부 서버로 전송되지 않습니다
- 네트워크 연결 없이 완전 오프라인으로 동작합니다
