# Image Chooser - 인물 사진 선별 앱

## 개요

여러 장의 비슷한 인물 사진 중에서 가장 잘 나온 사진 3장을 자동으로 선별해주는 iOS 앱.

## 타겟 플랫폼

- iOS 16.0+
- iPhone

## 기술 스택

- Swift / SwiftUI
- Apple Vision Framework (얼굴 감지 및 품질 분석)
- Core Image (이미지 품질 분석)
- PhotosUI (사진 선택)

## 핵심 기능

### 1. 사진 선택

- 사용자가 앨범에서 여러 장의 사진을 선택할 수 있다
- PhotosPicker를 사용하여 다중 선택 지원
- 최소 4장 이상 선택 시 분석 가능

### 2. 사진 분석

선택된 사진들을 다음 5가지 기준으로 평가한다:

| 기준 | 가중치 | 설명 | 분석 방법 |
|------|--------|------|-----------|
| 선명도 | 25% | 이미지가 흔들리지 않고 선명한지 | CIEdges 필터로 엣지 강도 측정 |
| 얼굴 품질 | 25% | 얼굴이 잘 캡처되었는지 | VNDetectFaceCaptureQualityRequest |
| 눈 뜨임 | 25% | 눈을 감지 않았는지 | 얼굴 랜드마크의 눈 영역 종횡비 분석 |
| 노출 | 15% | 밝기가 적절한지 (과다/부족 노출 감지) | CIAreaAverage로 평균 밝기 측정 |
| 구도 | 10% | 얼굴 위치가 적절한지 (중앙 또는 삼등분 법칙) | 얼굴 바운딩 박스 위치 분석 |

- 각 기준은 0~1 사이의 점수로 정규화
- 가중 합산하여 총점 산출
- 얼굴이 감지되지 않은 사진은 얼굴 관련 점수가 0으로 처리되어 자연스럽게 후순위로 밀림
- 분석 시 성능을 위해 이미지를 1024px로 리사이즈 후 처리

### 3. 결과 표시

- 총점 기준 상위 3장을 선정하여 순위와 함께 표시
- 각 사진의 항목별 점수를 시각적으로 표시 (바 차트)
- 나머지 사진들도 점수와 함께 목록으로 확인 가능

## 화면 구성

### 메인 화면

1. **사진 선택 영역**: 사진 선택 버튼, 선택된 사진 수 표시
2. **분석 버튼**: 선택 완료 후 분석 시작
3. **진행률 표시**: 분석 중 몇 번째 사진을 처리 중인지 표시
4. **결과 영역**:
   - 상위 3장: 큰 카드 형태로 순위, 사진, 총점, 항목별 점수 표시
   - 나머지: 작은 그리드로 사진과 점수 표시

```
    ├── ContentView.swift          # 메인 화면
    └── ResultCardView.swift       # 결과 카드 컴포넌트
```
